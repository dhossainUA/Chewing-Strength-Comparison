# Chewing-Strength-Comparison
Repository for Chewing Strength Estimation performance comparison between different chewing sensor modalities (EMG, Piezo-elctric, Piezo-resistive, Ear Canal Pressure Sensor). Applied ANOVA and Tukey's test to get the statistical inference
